# portfolio.com

Personal portfolio site for **Salem Ebrahim** — built with React, TypeScript, and Vite.

## Scripts

- `npm install` — install dependencies  
- `npm run dev` — local development  
- `npm run build` — production build  
- `npm run preview` — preview the build  

## Deploy

GitHub Actions deploys to GitHub Pages when you push to `main` (see `.github/workflows/deploy.yml`).
